# MarzbanX

MarzbanX is a fork of [Gozargah/Marzban](https://github.com/Gozargah/Marzban). This fork focuses on controller-managed node protocols, one-command node provisioning, HY2/AnyTLS with sing-box, and the Rust [MarzbanX-node](../MarzbanX-node) runtime.

The full README is currently maintained in:

- [English](./README.md)
- [简体中文](./README-zh-cn.md)

## Links

- Original upstream: [Gozargah/Marzban](https://github.com/Gozargah/Marzban)
- Rust node runtime: [MarzbanX-node](../MarzbanX-node)

## Note

Upstream donation addresses were removed from this fork README. Runtime compatibility names such as `marzban`, `marzban-cli`, and `/var/lib/marzban` may still appear in code and deployment paths until they can be renamed safely.
